获取某指数成分股¶

lhdata.pool.get_memb(index: str, day: str = None, start: str = None, end: str = None, db_config: dict = None, tb_name: str = None, hk: bool = False, **kwargs) → pandas.core.frame.DataFrame[source]¶

获取指数成分股。

  • 沪深300、中证500 由今日投资维护。
  • 创业板指、恒指、国企指数、道指、纳斯达克100、标普500、日经225 由Lighthorse维护。
Parameters:
  • index – 指数名字，如’hs300’. 支持上述6个指数，可以使用中、英、别名。建议使用指数对应的ticker，如hs300对应SHSZ300 Index。
  • day – 指定日期，会忽略start、end的输入。若day、start、end都为空，则设置为查询当日。注意此时当日数据可能还未准备好。
  • start – 起始日期（含），格式如‘2019-01-01’；空(‘’ 或 None) 则自动设置为1992-01-01。
  • end – 终止日期（含），格式如‘2019-09-01’；空(‘’ 或 None) 则自动设置为查询当日。
  • db_config – 指定 成分表 数据库连接，None则自动分配。
  • tb_name – 指定 成分表 的表名，格式为{db_name}.{tb_name}，None则自动分配。
  • hk – 在香港使用需设置为True。
  • kwargs – 其他get_fields支持传入的参数。
Returns:

DataFrame
