Alpha features¶

lhdata.features.get_alpha(symbol: Union[str, list, tuple] = '', day: str = None, start: str = None, end: str = None, index_ref: str = 'hs300', fields: Union[str, list, tuple] = '*', db_config: dict = None, tb_name: str = None, area: str = None, hk: bool = False, fmt: str = 'df', **kwargs)[source]¶

获取Alpha Features。

Parameters:
  • symbol – 如‘000001 CH Equity’；空(‘’ 或 [] 或 () ) 代表所有股票；支持股票列表或股票元组。
  • start – 起始日期（含），格式如‘2019-01-01’；空(‘’ 或 None) 则自动设置为1992-01-01。
  • end – 终止日期（含），格式如‘2019-09-01’；空(‘’ 或 None) 则自动设置为查询当日。
  • day – 指定日期，等效于设置了start=end=day, 会忽略start、end的输入。
  • index_ref – A股：[‘hs300’, ‘zz500’, ‘gemb]; US: [‘indu’, ‘spx’, ‘ndx’]; HK: [‘hsi’, ‘hscei’]; JP: [‘nky’]
  • fields – 指定’datetime’、’symbol’以外的字段，默认所有字段。
  • db_config – 指定 特征 数据库连接，None则自动分配。
  • tb_name – 指定 特征 表名，格式为{db_name}.{tb_name}，None则自动分配。
  • area – in [‘CH’, ‘US’, …]。指定symbol区域，不指定则根据symbol（或symbol list的第一个）判定。为空时，自动设置为’CH’。
  • hk – 在香港使用需设置为True。
  • fmt – 返回数据类型，支持DataFrame、DataFrame list、DataFrame dict. 默认为’df’。
  • kwargs – 其他lhdata.dbgetter.get_fields支持传入的参数。
Returns:

DataFrame 或 DataFrame list 或 DataFrame dict
