基础信息模块 lhdata.fund¶

  • A股由今日投资维护
  • 外盘由Lighthorse维护

获取股票列表¶

lhdata.fund.get_symbols(area: str = 'ch', active: bool = True, type_: str = 'A', board: str = None, order_by_symbol: bool = True, start: str = None, end: str = None, day: str = None, db_config: dict = None, tb_name: str = None, hk: bool = True, **kwargs) → pandas.core.frame.DataFrame[source]¶

获取某个市场所有的（外盘为订阅的）股票信息。

Parameters:
  • area – in [‘ch’, ‘us’, ‘hk’, ‘jp’]。
  • active – 是否正常交易。A股判断标准是退市日期为空，外盘通过market_status确认。
  • type – 品种代码, in [‘A’, ‘B’], None则包括A和B股。只对A股有效。
  • board – 板块代码, in [‘MAIN’(主板), ‘SME’(中小板), ‘GEM’(创业板), ‘STAR’(科创板)], None则所有版块。只对A股有效。
  • order_by_symbol – True -> 按股票代码排序, False -> 按上市日期排序。
  • start – 只返回该日期及之后上市的股票。None则不做限制。
  • end – 只返回该日期及之前上市的股票。None则不做限制。
  • day – 只返回该日上市的股票。None则不做限制。
  • db_config – 指定行情数据库连接，None则自动分配。
  • tb_name – 指定行情表名，格式为{db_name}.{tb_name}，None则自动分配。
  • hk – 在香港使用需设置为True。
  • kwargs – 其他dbgetter.get_fields支持传入的参数
Returns:

DataFrame

获取ipo日期¶

lhdata.fund.get_ipo_day(symbol: Union[str, list, tuple], db_config: dict = None, tb_name: str = None, area: str = None, hk: bool = True, **kwargs) → pandas.core.frame.DataFrame[source]¶

获取ipo日期。注意外盘股票有可能上市日期为空。

Parameters:
  • symbol – 如‘000001 CH Equity’；只支持单只股票，或股票列表(元组)。
  • db_config – 指定行情数据库连接，None则自动分配。
  • tb_name – 指定行情表名，格式为{db_name}.{tb_name}，None则自动分配。
  • area – in [‘CH’, ‘US’, …]。指定symbol区域，不指定则根据symbol（或symbol list的第一个）判定。为空时，自动设置为’CH’。
  • hk – 在香港使用需设置为True。
  • kwargs – 其他dbgetter.get_fields支持传入的参数
Returns:

DataFrame
