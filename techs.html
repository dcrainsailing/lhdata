技术指标模块 lhdata.techs¶

  • 支持talib技术指标

获取数据¶

lhdata.techs.get_data(symbol: Union[str, list], area: str = None, day: str = None, start: str = None, end: str = None, feature: Union[str, list] = None, tech: Union[str, list] = None, price: Union[str, list] = ('open', 'high', 'low', 'close', 'volume'), hk: bool = True, fmt: str = 'df', **kwargs)[source]¶

从数据库中获取数据，支持技术指标（需要安装talib）。

Parameters:
  • symbol – 如‘000001 CH Equity’；支持股票列表或股票元组，不支持为空。
  • area – in [‘CH’, ‘US’, …]。指定symbol区域，不指定则根据symbol（或symbol list的第一个）判定。为空时，自动设置为’CH’。
  • start – 起始日期（含），格式如‘2019-01-01’；空(‘’ 或 None) 则自动设置为1992-01-01。
  • end – 终止日期（含），格式如‘2019-09-01’；空(‘’ 或 None) 则自动设置为查询当日。
  • day – 指定日期，等效于设置了start=end=day, 会忽略start、end的输入。
  • feature – 如’WF0010’； 支持特征列表；’WF’代表所有Wave Feature。可以通过 lhdata.dispatcher.feature_col_dict 查询支持的feature系列。
  • tech – 如’atr_14’，代表计算14天ATR，参数通过下划线依次指定；支持多个技术指标，通过逗号隔开，或者使用列表。
  • price – 价格包含的列，默认不包含turnover。
  • hk – 在香港使用需设置为True。
  • fmt – 返回数据类型，支持DataFrame、DataFrame list、DataFrame dict. 默认为’df’。
  • kwargs – 其他 lhdata.prices.get_px 或者 lhdata.features.get_xxx 支持的参数。
Returns:

DataFrame 或 DataFrame list 或 DataFrame dict

计算技术指标¶

lhdata.techs.get_tech(df: pandas.core.frame.DataFrame, tech_string: str = None, func: talib.abstract.Function = None)[source]¶

计算技术指标。

Parameters:
  • df – 股票数据，需要包含计算技术指标所需的列。
  • tech_string – 如’atr_14’，代表计算14天ATR，参数通过下划线依次指定；与func至少有一个需要不为None。
  • func – talib的函数，None则通过tech_string自动解析；优先级>tech_string。
Returns:

DataFrame
