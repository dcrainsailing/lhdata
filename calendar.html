日历模块 lhdata.calendar¶

  • 从Bloomberg下载的除周末以外的假期，标记为holiday。
  • 非holiday，但数据库中无数据，判断为临时停牌，标记为temp。

获取某市场的交易日历¶

lhdata.calendar.get_calendar(area: str = 'CH', start: str = None, end: str = None, db_config: dict = None, db_name: str = None, tb_name: str = None, holiday_only: bool = False, fmt: str = 'index', hk: bool = False, **kwargs)[source]¶

指定市场，获得该市场的交易日历。

Parameters:
  • area – in [‘CH’, ‘US’, ‘HK’, ‘JP’]. 默认为’CH’. 支持大小写混用。
  • start – 起始日期（含），格式如‘2019-01-01’；空(‘’ 或 None) 则自动设置为1992-01-01。
  • end – 终止日期（含），格式如‘2019-09-01’；空(‘’ 或 None) 则自动设置为查询当日。
  • db_config – 指定 交易日历 数据库连接，None则自动分配。
  • db_name – 指定 交易日历 数据库名，None则自动分配。
  • tb_name – 指定 交易日历 表名，None则自动分配。
  • holiday_only – 只过滤假期，不过滤临时停牌则选择True。默认为False。
  • fmt – 返回格式。’index’ -> DatetimeIndex; ‘str’ -> string list; otherwise, Timestamp list.
  • hk – 在香港使用需设置为True。
  • kwargs – 其他get_fields支持传入的参数。
Returns:

DatetimeIndex 或 string list 或 Timestamp list

获取某市场某日的前一个交易日¶

lhdata.calendar.get_prev_day(day: str, area: str = 'ch', space: int = 30, db_config: dict = None, db_name: str = None, tb_name: str = None, hk: bool = False, **kwargs)[source]¶

指定市场，获得某日之前的一个交易日。通过查询交易日历实现。

Parameters:
  • day – 格式如‘2019-01-01’。
  • area – in [‘CH’, ‘US’, ‘HK’, ‘JP’]. 默认为’CH’. 支持大小写混用。
  • space – 搜索空间。默认为day之前30个自然日。
  • db_config – 指定 交易日历 数据库连接，None则自动分配。
  • db_name – 指定 交易日历 数据库名，None则自动分配。
  • tb_name – 指定 交易日历 表名，None则自动分配。
  • hk – 在香港使用需设置为True。
  • kwargs – 其他get_calendar支持传入的参数。
Returns:

date str or None

  • hk – 在香港使用需设置为True。
  • area – in [‘CH’, ‘US’, ‘HK’, ‘JP’]. 默认为’CH’. 支持大小写混用。
  • kwargs – 其他lhdata.dbgetter.get_prev_支持传入的参数。
Returns:

a list/tuple of symbols -> date str list; otherwise, date str
