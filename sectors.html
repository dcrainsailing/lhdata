行业模块 lhdata.sectors¶

  • A股由今日投资维护
  • 目前支持申万宏源行业体系
  • 外盘正在建设

获取某股票所属行业¶

lhdata.sectors.get_sector(symbol: Union[str, list], system: str = None, level: int = 3, db_config: dict = None, df_static: pandas.core.frame.DataFrame = None, progress: bool = False, **kwargs) → pandas.core.frame.DataFrame[source]¶

给定股票，查询所属行业。

Parameters:
  • symbol – 如‘000001 CH Equity’. 支持股票列表。
  • system – 行业体系，in [‘gics’, ‘exchange’, ‘investoday’, ‘swhy’, ‘citic’]。None则自动分配： A股swhy， 外盘gics。
  • level – 如果超出该体系的级别范围，返回该体系所有级别的对应信息
  • db_config – 指定数据库连接，None则自动分配。
  • df_static – 存储好的股票、代码表
  • progress – 显示下载进度条。
  • kwargs – 其他shadow_split支持传入的参数
Returns:

DataFrame， 包含列：symbol、name、code_sector, 或 DataFrame list

其他接口¶

lhdata.sectors.get_sector_table(system: str = None, max_lv: int = None, transform: bool = True, area: str = 'ch', db_config: dict = None, tb_name: str = None, static: bool = False) → pandas.core.frame.DataFrame[source]¶

指定体系的行业分类代码表。由今日投资维护。

Parameters:
  • system – 行业体系，in [‘gics’, ‘exchange’, ‘investoday’, ‘swhy’, ‘citic’]。None则自动分配： A股swhy， 外盘gics。
  • max_lv – 最大到什么级别，None为不限制。
  • transform – 查询到的表各个级别的code混在一起，将表转换为更易读的形式。
  • area – in [‘ch’, ‘us’, ‘jp’, ‘hk’, ‘ovs’].
  • db_config – 指定数据库连接，None则自动分配。
  • tb_name – 指定表名，None则自动分配。
  • static – 是否使用已存储的副本。如果无法连接到服务器会自动使用。
Returns:

DataFrame

给定一个体系的行业代码，返回该行业涵盖的股票。

Parameters:
  • code – 行业代码，如110101。支持代码列表。
  • system – 行业体系，in [‘gics’, ‘exchange’, ‘investoday’, ‘swhy’, ‘citic’]。None则自动分配： A股swhy， 外盘gics。
  • db_config – 指定数据库连接，None则自动分配。
  • df_static – 存储好的股票、代码表
  • area – like ‘us’. 当system=’gics’的时候有用, 或用来确定system 。None则不对区域做限制。
  • progress – 显示下载进度条。
  • kwargs – 其他shadow_split支持传入的参数
Returns:

DataFrame，或 DataFrame list
