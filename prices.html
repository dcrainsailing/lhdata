行情模块 lhdata.prices¶

  • 数据库中保存两种数据：原始行情 和 复权数据 。
  • 查询前复权行情时，分别获取这两份数据，使用复权数据对原始行情实时修改。
  • 这样的好处是：数据一旦入数据库，除非有更新或校正，就无需更改。

获取行情¶

lhdata.prices.get_px(symbol: Union[str, list, tuple], day: str = None, start: str = None, end: str = None, fields: Union[str, list, tuple] = '*', px_db_config: dict = None, px_db_name: str = None, px_tb_name: str = None, adj_type: str = 'back', adj_all: bool = True, adj_db_config: dict = None, adj_tb_name: str = None, suspend: str = '', trim: bool = False, hk: bool = False, area: str = None, fmt: str = 'df', index_col: Union[str, list, tuple] = None, progress: bool = False, **kwargs)[source]¶

获取行情，默认为前复权。

支持：

  • 指定股票，或股票列表，或设置为空查询所有股票。
  • 指定某一日，或指定起止日期。
  • 指定字段。
  • 指定数据库读取行情，该db行情表结构需与161行情表一致，表名相同，且每五年分一个表。
  • 停牌处理，可选不包含、设置为空、使用最近价格填充。
  • 强制对齐到交易日历。
  • 指定市场。
  • 返回单个df、df列表或df字典。
  • 设置df的index。
Parameters:
  • symbol – 如‘000001 CH Equity’；空(‘’ 或 [] 或 () ) 代表所有股票；支持股票列表或股票元组。
  • start – 起始日期（含），格式如‘2019-01-01’；空(‘’ 或 None) 则自动设置为1992-01-01。
  • end – 终止日期（含），格式如‘2019-09-01’；空(‘’ 或 None) 则自动设置为查询当日。
  • day – 指定日期，等效于设置了start=end=day, 会忽略start、end的输入。
  • fields – 指定字段，如’close’ 或 [‘open’, ‘close’]; 会自动添加‘datetime’和‘symbol’两个字段。默认为所有字段。
  • px_db_config – 指定 原始行情 数据库连接，None则自动分配。
  • px_db_name – 指定 原始行情 数据库名，None则自动分配。
  • px_tb_name – 指定 原始行情 表名，格式为{db_name}.{tb_name}，None则自动分配。
  • adj_type – 复权方法。’back’ -> 前复权；None -> 不复权。 默认为’back’.
  • adj_all – 若复权不依赖所指定的时间段，则设置为True。仅复权该时间段，则设置为False. 当指定day时，adj_all强制为True。
  • adj_db_config – 指定 复权数据 数据库连接，None则自动分配。
  • adj_tb_name – 指定 复权数据 表名，格式为{db_name}.{tb_name}，None则自动分配。
  • suspend – 停牌处理。’’ -> 不包括停牌；’nan’/’null’ -> 填充nan；’px’ -> 填充最近的有效价格。默认为’‘。
  • trim – 是否强制对齐到同一个交易日历。不强制时，指定的start之后上市的股票只对齐到上市之后的交易日历。
  • hk – 在香港使用需设置为True。
  • area – in [‘CH’, ‘US’, …]。指定symbol区域，不指定则根据symbol（或symbol list的第一个）判定。为空时，自动设置为’CH’。
  • fmt – 返回数据类型，支持DataFrame、DataFrame list、DataFrame dict. 默认为’df’。
  • index_col – 返回的DataFrame设置index，对list、dict也有效。
  • progress – 显示下载进度条。
  • kwargs – 其他选项。 nbr_thread: 使用多少个进程。0代表自动；1或负值代表不使用多进程。 chunksize：每个进程处理多少个查询。默认为1。
Returns:

DataFrame 或 DataFrame list 或 DataFrame dict

查询某支股票的日涨跌幅。

Parameters:

  • db_config – 指定行情数据库连接，None则自动分配。
  • db_name – 指定行情数据库名，None则自动分配。
  • tb_name – 指定行情表名，格式为{db_name}.{tb_name}，None则自动分配。
  • incl_abs – 是否包含涨跌额
  • hk – 在香港使用需设置为True。
  • area – in [‘CH’, ‘US’, …]。指定symbol区域，不指定则根据symbol（或symbol list的第一个）判定。为空时，自动设置为’CH’。
  • kwargs – 其他get_px支持传入的参数。

DataFrame
